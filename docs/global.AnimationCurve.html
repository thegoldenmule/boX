AnimationCurve

AnimationCurve

new AnimationCurve(keys) → {AnimationCurve}

Defines a continuous function through points in the unit interval on R^2. These points are given as AnimationCurveKeys.
Parameters:
Name Type Description
keys Array An optional array of Number to populate the AnimationCurve. There should be an even number of floats, each pair representing an (x, t) point.
